##APP STAGE
Clone the project

in the Client/ directory run:
to install modules
#npm install
then
#npm start

in the Server/ directory, also run:
to install modules
#npm install
to start the node server
#npm run start:dev

in the apiLogin/:
to install all requirements
#pip install -r requirements.txt
to run the server:
#uvicorn app.main:app --reload
